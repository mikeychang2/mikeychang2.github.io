November 9, 2014

Classes are first-class objects, each of which are instances of the class Class. Once created, you can set methods and attributes so that whenever a new instance of that class is created, it has contained within it a set of properties via instance variables and methods. Basically, you want to be able to create many objects with some common variables and accessible methods, but you don’t want to have to recreate these objects all individually.

Class Methods vs. Instance Methods

Within the class, you can create different methods to call on each instance of the class. For instance, you can create a simple “greeting” method that spits up a string introducing the person - which you would call on an instance of the class, not the class itself. This is an instance method.
